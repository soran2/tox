This repository contains the following files:

1. featurizer.py 
    This module contains functions for featurizing molecules given SMILES strings.

2. models.py
    This module contains functions for machine learning models. 

3. Project_2.ipynb
    A notebook containing basic procedure for calling the functions in featurizer.py and 
    models.py to predict molecular toxicity with machine learning models.

4. data.csv
    File containing toxicity data.

5. Report.pdf
    Short report explaining the results.
    
6. README.txt
    This file.
